# Face Recognition Attendance System

## 📌 Overview
This project is an AI-based attendance system that uses face recognition to automatically mark attendance in real-time. It eliminates manual attendance and improves accuracy using machine learning techniques.

---

## 🚀 Features
- Real-time face detection and recognition
- Automated attendance marking
- Machine Learning-based face classification
- Efficient image processing pipeline
- User-friendly interface (Streamlit)

---

## 🛠️ Technologies Used
- Python
- OpenCV
- Scikit-learn
- NumPy & Pandas
- Streamlit

---

## ⚙️ How It Works
1. Capture images of users
2. Extract facial features (embeddings)
3. Train machine learning model
4. Recognize faces in real-time
5. Mark attendance automatically

---

## ▶️ How to Run
1. Run dataset creation script  
2. Run preprocessing script  
3. Train the model  
4. Run face recognition script  

---

## 📸 Output
- Detects faces in real-time  
- Identifies known users  
- Marks attendance automatically  


---
